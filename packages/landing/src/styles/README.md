# Styles

CSS modules and stylesheets for the Forj landing page.

## Structure (Stack 3)

- `main.css` - Base styles, CSS variables, utilities ✅
- `nav.module.css` - Navigation styles
- `hero.module.css` - Hero section styles
- `terminal.module.css` - Terminal animation styles
- `features.module.css` - Feature grid styles
- `pricing.module.css` - Pricing card styles
- `footer.module.css` - Footer styles

CSS Modules allow component-scoped styling to avoid conflicts.
