# Components

UI components for the Forj landing page.

## Planned Components (Stack 3)

- `Nav.ts` - Navigation header
- `Hero.ts` - Hero section with CTA
- `Terminal.ts` - Animated terminal demo
- `Features.ts` - Feature grid
- `Pricing.ts` - Pricing cards
- `WaitlistForm.ts` - Email signup form (Stack 4)
- `Footer.ts` - Footer section

Each component will be a TypeScript module that returns an HTML string or DOM element.
